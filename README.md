**Malware Analysis** is malicious software. I never knew the acronym before. Malware comes in a few types like virus, Trojan Horse, and Ransomeware

- Virus is a piece of code attached to a program and spreads from one computer to another making computers unusable

- Trojan horse is a program that hides malicious functions like giving a back door access to your computer

- Ransomware is a program that makes files uncreadable without encryption password and ask for ransom to get the passward to see contents in that file

Static analysis is inspecting the program language without running it

Dynamic analysis is running malware in a controlled inviroment tike a VM, to see its behavior
